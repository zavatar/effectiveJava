Effective Java
==============

Build
-----

we use gradle to build this project. So, if you have gradle:

- gradle build

Note: you should modify the protoc path in protobuf.compiler to your own path

Otherwise:

- ./gradlew build

you can also import the project to IntelliJ IDEA

Dependencies
------------

- lombok 1.14.2
- junit 4.+
- protobuf 2.6.1
